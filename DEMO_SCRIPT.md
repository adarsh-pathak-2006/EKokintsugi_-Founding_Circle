# Demo Video Script

Use this sequence for the end-to-end walkthrough:

1. Open the app and show the landing/login page.
2. Login as admin with `admin@ekokintsugi.com / admin123`.
3. Show:
   - user table
   - review tracking
   - return tracking
   - points management
4. Open one seeded pilot user detail through the user table context and mention:
   - assigned shoe
   - symbolic tree
   - points
   - review progress
5. Logout.
6. Login as `pilot1@ekokintsugi.com / pilot123`.
7. Show the user dashboard:
   - shoe details
   - tree details
   - QR code
   - progress bar
8. Open the QR landing page and resize to mobile width.
9. Return to the dashboard and submit a weekly review.
10. Show the dashboard updated with the new review count.
11. Open the return request page and submit a return request.
12. Logout and login as admin again.
13. In return tracking, update the request status and assign final action:
   - Requested
   - Received
   - Completed
   - Repair / Reuse / Recycle
14. In points management, add and approve points for a user.
15. End by summarizing the circular lifecycle flow:
    Scrap → Shoe → User → Review → Return → Recycle
